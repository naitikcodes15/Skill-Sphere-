# SkillSphere

SkillSphere is a professional real-time 1v1 coding challenge and skill assessment platform. It enables developers to engage in live competitive programming battles and interactive practice quizzes in a synchronized environment.

Link to website: [https://xskillsphere.vercel.app](https://xskillsphere.vercel.app)

---

## Key Features

- [Real-time 1v1 Battle Arena] Compete in live coding matches with synchronized state, opponent tracking, and automatic evaluation.
- [Interactive Quizzes] Test your knowledge across multiple categories (Linux, Programming, DevOps) with auto-grading.
- [Built-in Execution Engine] High-performance code runner supporting C, C++, Java, Python, and JavaScript.
- [Integrated Monaco Editor] Professional-grade editing experience with syntax highlighting and automatic starter code generation.
- [Secure Auth & Socials] Firebase-powered user accounts, real-time friend search, and live chat.

## Tech Stack

- [Frontend] React 19 (TypeScript), Vite, TailwindCSS, Monaco Editor.
- [Backend] Node.js (TypeScript), Express, Socket.io, Mongoose/MongoDB.
- [Database & Auth] Firebase Authentication & Firestore, MongoDB Atlas.
